== Installation ==

You can install Braintree Donations directly from the WordPress admin panel:

	1. Visit the Plugins > Add New and search for 'Faster Pagination'.
	2. Click to install.
	3. Once installed, activate and it is functional.
	
OR

Manual Installation:

	1. Download the plugin, then extract it.
	2. Upload `ft-pagination` extracted folder to the `/wp-content/plugins/` directory
	3. Activate the plugin through the 'Plugins' menu in WordPress
	
You're done! The Plugin is ready to use, for more please check the plugin description.
